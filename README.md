

# 游戏选题

游戏名：守机快跑

2D横版像素风游戏

高随机性、短流程的多人聚会游戏

## 游戏概述

玩家们扮演一个私家侦探团体，一名玩家手持手机。

手机属于一个黑社会老大，其中藏有一些指向其犯罪证据的线索，会不断发送GPS信息，被该黑社会团体收到。黑社会喽啰们根据此追杀玩家，想要取回手机。

玩家要做的，是合作保护好手机，活到后援抵达，同时收集足够多的证据，以便扳倒黑老大、赢取奖赏。

## 玩法

* 合作策略：一名玩家会不断吸引火力，其他玩家需要保护这名玩家。
* 生存系统：消耗物资、杀死敌人（射击）
* 任务系统：即收集黑老大的犯罪线索，关联最终结局
